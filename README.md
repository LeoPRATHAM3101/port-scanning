# 📡 Port Scanning using Nmap

## 🚀 Overview
This project demonstrates practical network reconnaissance using **Nmap**, a widely used cybersecurity tool. The goal is to identify open ports, detect running services, and understand how attackers gather information about target systems.

---

## 🎯 Objectives
- Perform port scanning on a target system
- Identify open and filtered ports
- Detect running services and versions
- Understand basic enumeration techniques used in penetration testing

---

## 🛠️ Tools & Technologies
- Nmap
- Terminal (macOS / Kali Linux)

---

## 🔍 Methodology

The following command was used to perform the scan:

```bash
nmap -sS -sV -O scanme.nmap.org
